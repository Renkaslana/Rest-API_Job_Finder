üìö API Endpoints

üîç Job Search & Listing

GET /api/jobs
‚ú® ENHANCED! Recommended jobs untuk Home screen dengan filter options
Query Parameters:
‚Ä¢ limit - Number of jobs (default: 30, max: 100)
‚Ä¢ sort - Sort order: latest (default), salary
‚Ä¢ salary - Filter only jobs with salary (true/false)

Example:
/api/jobs?limit=20&sort=salary&salary=true
GET /api/search
‚ú® ENHANCED! Advanced search & filter dengan pagination
Query Parameters:
‚Ä¢ q - Keyword search (title, company, description)
‚Ä¢ category - Filter by category (IT, Marketing, Design, etc)
‚Ä¢ location - Filter by location (Jakarta, Bandung, etc)
‚Ä¢ salaryMin - Minimum salary (numeric)
‚Ä¢ jobType - Job type (full-time, part-time, contract, internship)
‚Ä¢ sort - Sort: latest, salary, relevance
‚Ä¢ page - Page number (default: 1)
‚Ä¢ limit - Results per page (default: 30, max: 100)

Example:
/api/search?q=developer&category=IT&salaryMin=5000000&sort=salary
GET /api/filters
Get available filter options (categories & locations)
GET /api/job?url=<jobstreet_url>
Get full job detail dengan deskripsi lengkap
Example:
/api/job?url=https%3A%2F%2Fid.jobstreet.com%2Fid%2Fjob%2F...

üíª Cara Penggunaan

JavaScript / Fetch API

// List semua jobs
fetch('/api/jobs')
  .then(res => res.json())
  .then(data => console.log(data.data.jobs));

// üÜï Search dengan filter
fetch('/api/search?q=developer&category=IT&location=Jakarta')
  .then(res => res.json())
  .then(data => console.log(data.data.jobs));

// üÜï Get available filters
fetch('/api/filters')
  .then(res => res.json())
  .then(data => {
    console.log('Categories:', data.data.categories);
    console.log('Locations:', data.data.locations);
  });

// Detail 1 job
const jobUrl = encodeURIComponent('https://id.jobstreet.com/id/job/12345678');
fetch(`/api/job?url=${jobUrl}`)
  .then(res => res.json())
  .then(data => console.log(data.data));

Python

import requests

# List jobs
response = requests.get('/api/jobs')
jobs = response.json()['data']['jobs']

# üÜï Search dengan filter

response = requests.get('/api/search', params=params)
filtered_jobs = response.json()['data']['jobs']

# üÜï Get available filters
filters = requests.get('/api/filters').json()['data']
print('Categories:', [c['name'] for c in filters['categories']])
print('Locations:', filters['locations'])

# Detail job
from urllib.parse import quote
job_url = quote('https://id.jobstreet.com/id/job/12345678')
detail = requests.get(f'/api/job?url={job_url}')
print(detail.json()['data'])
