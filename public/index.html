üîç Job Search & Listing

GET /api/jobs
üìã Recommended jobs untuk Home screen (simple, no filters)
Query Parameters:
‚Ä¢ limit - Number of jobs (default: 30, max: 100)
‚Ä¢ page - Page number (default: 1)

‚ö†Ô∏è Not Supported:
‚Ä¢ sort, salary, category, location, q
‚Ä¢ Use /api/search for advanced filters
‚Ä¢ Use /api/jobstreet for recommendations

Example:
/api/jobs?limit=20&page=1
GET /api/search
üîç PRODUCTION-READY! Search jobs by location with auto-normalization
Query Parameters:
‚Ä¢ location - REQUIRED - Any Indonesia region (auto-normalized)
  Examples: "banten", "Jawa Tengah", "DI Yogyakarta"
‚Ä¢ classification - Optional job category slug
  Examples: "banking-financial-services", "it-technology"
‚Ä¢ page - Page number (default: 1, min: 1)
‚Ä¢ limit - Results per page (default: 20, max: 30)

‚úÖ Location Auto-Normalization:
‚Ä¢ "Jawa Tengah" ‚Üí "jawa-tengah"
‚Ä¢ "DI Yogyakarta" ‚Üí "yogyakarta"
‚Ä¢ "Nusa Tenggara Barat" ‚Üí "nusa-tenggara-barat"
‚Ä¢ Works for ALL Indonesia regions!

Pagination:
‚Ä¢ Response includes hasNextPage
‚Ä¢ Perfect for infinite scroll

üö´ Not Supported:
‚Ä¢ sort, salary, posted parameters
‚Ä¢ Invalid params ‚Üí 400 error (not 500)

Examples:
/api/search?location=banten
/api/search?location=Jawa Tengah&classification=it-technology
/api/search?location=jakarta&page=2&limit=20
GET /api/jobs/:jobId NEW
üìÑ RECOMMENDED! Get FULL job detail menggunakan jobId (stabil & konsisten)
Path Parameter:
‚Ä¢ jobId - Job ID dari JobStreet (6-10 digit numeric, REQUIRED)

Response Fields:
‚Ä¢ jobId - Job ID
‚Ä¢ title - Job position title
‚Ä¢ company - Company name
‚Ä¢ location - Job location
‚Ä¢ salary - Salary info (if available)
‚Ä¢ postedLabel - When the job was posted
‚Ä¢ description - Array of description paragraphs
‚Ä¢ requirements - Array of job requirements
‚Ä¢ applyUrl - JobStreet URL untuk apply

‚úÖ Advantages:
‚Ä¢ Tidak perlu URL encoding
‚Ä¢ jobId sederhana (string numeric)
‚Ä¢ Error handling jelas (400/404/500)
‚Ä¢ Lebih stabil & konsisten

Examples:
/api/jobs/89023836
/api/jobs/75934159

üí° How to use: Extract jobId dari response job list, lalu gunakan untuk detail
GET /api/job?url=<jobstreet_url> LEGACY
üìÑ Get FULL job detail dengan URL lengkap (masih didukung, tapi gunakan /api/jobs/:jobId untuk pengalaman lebih baik)
Query Parameters:
‚Ä¢ url - Full JobStreet job URL (REQUIRED, URL-encoded)

‚ö†Ô∏è Note: Endpoint ini masih berfungsi untuk backward compatibility, tapi disarankan menggunakan /api/jobs/:jobId yang lebih stabil.

JavaScript / Fetch API

// List semua jobs
fetch('/api/jobs')
  .then(res => res.json())
  .then(data => console.log(data.data.jobs));

// Flexible search by location
fetch('/api/search?location=banten')
  .then(res => res.json())
  .then(data => console.log(data.jobs));

// Search by location + classification
fetch('/api/search?location=jawa-tengah&classification=information-technology')
  .then(res => res.json())
  .then(data => console.log(data.jobs));

// JobStreet recommendations
fetch('/api/jobstreet?page=1&limit=20')
  .then(res => res.json())
  .then(data => console.log(data.data));

// Detail 1 job (NEW - Recommended)
const jobId = '89023836'; // Extract from job list response
fetch(`/api/jobs/${jobId}`)
  .then(res => res.json())
  .then(data => console.log(data));

// Detail 1 job (Legacy - Still supported)
const jobUrl = encodeURIComponent('https://id.jobstreet.com/id/job/12345678');
fetch(`/api/job?url=${jobUrl}`)
  .then(res => res.json())
  .then(data => console.log(data.data));

// Articles list
fetch('/api/articles?limit=5')
  .then(res => res.json())
  .then(data => console.log(data.articles));

// Article detail
fetch('/api/articles/tips-interview-kerja-sukses')
  .then(res => res.json())
  .then(data => console.log(data));

Python

import requests

# List jobs
response = requests.get('/api/jobs')
jobs = response.json()['data']['jobs']

# Flexible search by location
response = requests.get('/api/search?location=banten')
jobs = response.json()['jobs']

# Search with location + classification
response = requests.get('/api/search', params={
    'location': 'jawa-tengah',
    'classification': 'information-technology'
})
jobs = response.json()['jobs']

# JobStreet recommendations
jobstreet = requests.get('/api/jobstreet?page=1&limit=20')
recommendations = jobstreet.json()['data']

# Detail job (NEW - Recommended)
job_id = '89023836'  # Extract from job list response
detail = requests.get(f'/api/jobs/{job_id}')
print(detail.json())

# Detail job (Legacy)
from urllib.parse import quote
job_url = quote('https://id.jobstreet.com/id/job/12345678')
detail = requests.get(f'/api/job?url={job_url}')
print(detail.json()['data'])

# Articles list
articles = requests.get('/api/articles?limit=5')
print(articles.json()['articles'])

# Article detail
article = requests.get('/api/articles/tips-interview-kerja-sukses')
print(article.json())
